Registrations {#registration}

Register for this message workflow by sending an SMS with the following format: R Patient Name

For example, to register "Maria Soleil" you would send: R Maria Soleil

Visits

Visits are recorded by sending an SMS in the format V 12345, replacing 12345 with the Patient's ID.

Deliveries
